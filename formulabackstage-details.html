Formula Backstage

Project information

  • Language: Python
  • APIs: FastF1, Tkinter, Matplotlib, MPLCursors...
  • Team: Mario Galan Sobrino
  • Features: Detailed graphs, hover annotations, details for individual laps
  • Project date: June 2025 - Ongoing project

Formula Backstage is a Formula 1 telemetry app made using Python and multiple APIs such as: FastF1, Tkinter, Custom Tkinter, Matplotlib and MPLCursors.

This project allows users to load the telemetry of F1 drivers across an F1 session. The main chart will display selected drivers laptimes and compounds used. Users then have the ability to select multiple laps, whihch will be displayed below the chart, and display detailed data of said laps. It is also possible to edit the color of the selected laps. The detailed graphs include: speed, throttle input, brake input, gears, RPM, delta and track dominance layout. The graphs include hover annotations to see data of specific points throughout the lap.
