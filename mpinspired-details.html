Metroid Prime

Project information

  • Engine: Unreal Engine 4
  • Team: Mario Galan Sobrino & Sebastian Valdes Sanchez
  • Role: Gameplay and AI programmer
  • Features: Basic AI enemies, particles, events, audio, animations...
  • Project date: January 2022 - May 2022

This project was developed for one of the modules of my second HND in computing year using Unreal Engine 4. This was my first big project using Unreal Engine 4.

This project was developed alongside one of my classmates called Sebastian Valdes Sanchez. The project aims to recreate a small version of the original Metroid Prime game using our own programming experience. The game includes tons of tech to try and make it as close as possible to the original release. The game has patrolling enemies with basic AI behaviour and detection, lock on mechanic, multiple types of ammunition to use, faithful recreated movement, drops, destructible meshes to synergy with Samus gun, particles, sound effects...

Developing this game was very challenging since it was our first time developing a big and fully functional Unreal Engine 4 project. The result could not have been better for the experience and time we had in my opinion. This project taught me plenty of things about the engine, aspects that I will later use in future developments.

All the Metroid Prime assets used in this project are for academic purpose only. Ultimately, the owner of the assets is Nintendo.
