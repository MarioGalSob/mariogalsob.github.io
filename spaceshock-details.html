Teseo Bros

Project information

  • Engine: Unreal Engine 4
  • Team: Mario Galan Sobrino & Sebastian Valdes Sanchez
  • Role: Gameplay, AI and networking programmer
  • Features: Complex AI enemies, local multiplayer, Particles, Events, Sounds...
  • Project date: September 2022 - May 2023

Teseo Bros is a game that was developed for one of the modules of my final HND in computing year using Unreal Engine 4.

This project was developed alongside one of my classmates called Sebastian Valdes Sanchez. The project is about escaping a labyrinth while killing enemies and solving puzzles. the game features multiple tech such as fully customizable enemy spawners, advanced enemy AI using behaviour trees and other tools provided by Unreal, fully functional local multiplayer for up to 2 players, complex main character mechanics and movements, particles, sounds, animations...

The main roles I had within this project were the enemies AI and all the networking aspects of the game. Both tasks were quite challenging, especially the networking aspect, but it ended up working quite well in the end. I am very satisfied with my work done in this project and I am also very happy with all the new tech I learned since it has been quite helpful in other projects.

All the Metroid Prime assets used in this project are for academic purpose only. Ultimately, the owner of the assets is Nintendo.
