This is my latest project. The game is being developed using Unreal Engine 5.

The main goal of this project is to becomem more familiar with Unreal Engine 5 and to improve my programming skills even more. The game has a big focus on AI with two enemies with different behaviours and ways of dealing with the player. The game also has many mechanics that improve the gameplay loop making it more enjoyable.

So far this project is helping me a lot to get used to the new editor and features of Unreal Engine 5. On top it is allowing me to improve my programming skills even more.
